# SHONEN STRIKE (Anime Fighters)
Contribuidores: Cauã Guerreiro, Douglas Gobitsch e Vinicius Raiol.

Um jogo de luta 2D desenvolvido em Python utilizando a biblioteca **Pygame**. Escolha seus personagens favoritos dos animes mais populares e enfrente uma campanha em 3 fases desafiadoras.

## 🎮 Sobre o Jogo

O **Shonen Strike** possui um sistema de combate rápido e dinâmico inspirado nos clássicos jogos de luta. O jogo conta com um menu principal, tela de seleção de personagens e uma campanha onde você deve derrotar 3 chefes sequenciais, cada um possuindo sua própria IA (Inteligência Artificial).

### Características Principais
- **Sistema de Combate:** Cada personagem possui três tipos de ataques:
  - Ataque Leve (Rápido, menor dano)
  - Ataque Médio (Equilibrado, médio alcance)
  - Ataque Especial (Alto dano, inclui projéteis/magias e maior tempo de recarga)
- **Modo Campanha:** Enfrente 3 inimigos poderosos em cenários variados.
- **Gráficos:** Sprites incrivelmente detalhados, baseados nos modelos clássicos do Nintendo DS.

## 🥷 Personagens Jogáveis

- **Ichigo Kurosaki** (Bleach) - O Shinigami Substituto com sua poderosa Zangetsu.
- **Monkey D. Luffy** (One Piece) - O Capitão dos Piratas do Chapéu de Palha.
- **Naruto Uzumaki** (Naruto) - O Ninja hiperativo da Aldeia da Folha.
- **Sasuke Uchiha** (Naruto) - O último Uchiha obcecado por vingança.
- **Roronoa Zoro** (One Piece) - O exímio espadachim do estilo Santoryu.
- **Kenpachi Zaraki** (Bleach) - O sanguinário Capitão do 11º Esquadrão.

## 👹 Chefes da Campanha

1. **Sosuke Aizen** (Fase 1: Soul Society)
2. **Pain** (Fase 2: Templo Uchiha)
3. **Donquixote Doflamingo** (Fase 3: Base Marineford)

## 💻 Requisitos e Instalação

### Pré-requisitos
- **Python 3.x**
- **Pygame** (`pip install pygame`)

### Como Jogar
1. Faça o clone ou o download do repositório.
2. Certifique-se de ter as dependências instaladas.
3. No terminal, execute o arquivo principal:
   ```bash
   python main.py
   ```
